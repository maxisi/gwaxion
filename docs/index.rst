gwaxion
=======

**gwaxion** is a package to facilitate computations related to gravitational waves from ultralight-boson condensates around black holes. The code was primarilty developed in writing arXiv:1810.03812, but it is of broad applicability.

This documentation is being actively developed. Usage examples can be found :doc:`here <gallery>` and API documentation in the :ref:`modindex`.
